1. Overview

Tujuan Utama Aplikasi

Annotator Pro adalah asisten bot Telegram pintar berbasis kecerdasan buatan (AI) yang berfungsi untuk mengevaluasi teks dan gambar. Bayangkan seperti guru penilai otomatis—pengguna mengirimkan pekerjaan, dan bot otomatis memberi komentar atau perbaikan.

Untuk mencegah kebangkrutan server, bot ini memakai sistem "Sistem Koin" (Credit-Based Billing) dimana setiap kali bertanya ke AI, saldo koin kita otomatis terpotong.

User Flow (Alur Jalan)

  1. Chat & Daftar: Budi nge-chat bot dan diberi bonus koin.
  2. Pilih Menu: Budi memilih mau ngecek tulisan atau ngecek gambar.
  3. Pengolahan: Bot merapikan pesannya, menelpon otak AI.
  4. Balasan: Bot menerima hasil, koin Budi dipotong, jawaban muncul di HP Budi.

2. Cara Menjalankan Bot di Lokal

Catatan Penting: Pastikan kamu sudah menginstall Python 3.9+ di laptopmu agar bisa menjalankan bot ini.

Langkah Instalasi Mesin

  • Buka folder kode bot ini di Terminal/Command Prompt.
  • Bikin ruang kedap khusus biar tidak bentrok aplikasinya:
    python -m venv venv
  • Nyalakan ruang tertutup tadi:
    (Di Mac/Linux) source venv/bin/activate
    (Di Windows) venv\Scripts\activate
  • Download & pasang mesin pelengkap:
    pip install -r requirements.txt
  • Buka bot kesayanganmu:
    python bot.py

Persiapan File Wajib (.env)

Bot ini menolak menyala sebelum kamu bikin 1 file misterius pelindung password bernana .env. Isikan dengan:

TELEGRAM_TOKEN=123...abc (Minta tiket dari @BotFather di telegram)
DATABASE_URL=sqlite:///bot.db (Atau alamat cloud Postgres)
KIE_API_KEY=rahasianegara123 (Tiket buat nyambung ke otak AI)

Bila sewaktu-waktu bot macet tidak balas pesan, cepat buka Notepad dan lihat isi file bot_logs.txt karena di sanalah aplikasi menulis jeritan error terakhirnya.

3. Arsitektur Sistem

Perkenalan Tokoh Utama Kode Kita:

Dia yang ngurusin balasan chat "Hai, ini menu kami". Kalau mau ubah sapaan awal, edit orang ini.

Petugas matematika. Kalau mau rubah bonus koin saat daftar dari 500 jadi 1000, temui dia.

4. Relasi Database (Laci Penyimpanan)

Aplikasi ini memiliki 4 tabel utama. Bayangkan tabel ini sebagai Excel yang terhubung satu sama lain.

erDiagram USERS ||--o{ TRANSACTIONS : "punya bon belanja" USERS ||--o{ EVALUATIONS : "punya riwayat ketikan" PROJECTS ||--o{ TASKS : "memiliki anak menu" USERS { int user_id "ID Rahasia Telegram" int balance "Saldo Koin saat ini" } TRANSACTIONS { int amount "+Tambah / -Kurang" string type "topup/deduction" } EVALUATIONS { string ai_output "Jawaban AI" string feedback "Good/Bad pencetan" }

Tabel `users`

Buku Absen. Saat temanmu pertama kali chat, ID telegram dia, bahasa pilihan dia, dan koin dia tersimpan selamanya disini sampai dihapus.

Tabel `transactions`

Buku Laporan Kas. Isinya "Tgl 5: Budi potong 5 poin karena nanya PR" atau "Tgl 6: Budi di topup Bos 100 poin". Gunanya agar si Budi tidak bisa berbohong saldo koinnya hilang mendadak.

5. Fitur & Fungsi Utama

A. Mengecek Teks (PR & AFM)

Merupakan tugas dasar dimana bot dikasih bacaan dan disuruh meralat kesalahan gaya dan tata bahasanya.

  • Kode yang Mengurus: bot.py (cari `process_segmented_input`)
  • Kronologi Kejadian: User masuk ke Menu Teks ➔ Bot pinta Input teks ➔ Bot kirim kotak ke AI KIE ➔ Saldo terpotong ➔ Pesan balasan muncul.
🛠 Cara Rubah Kalau Mau Diedit: Buka bot.py, ketikan "process_segmented_input" di fitur pencarian. Di sana ada tulisan await update.message.reply_text("Berikut hasilnya"). Jika kata balasan itu mau diganti "Nih Bos Hasilnya", ubahnya disitu. Warning: AWAS salah hapus kode deduction, saldo bisa jadi tak pernah dipotong!

B. Menilai 4 Gambar Seri (VCG)

Fitur visual dimana bot disuruh lihat 4 gambar. Bot jadi seperti juri pencari bakat yang merangkum mana gambar yang sempurna dan ada cacat piksel.

  • Kode yang Mengurus: bot.py (cari `process_vcg_images`)
  • Kronologi Kejadian: User disuruh upload foto tahap 1 ➔ sistem catat ID ➔ Upload foto 2, 3, dan 4 ➔ sistem meremas 4 foto jadi Text Base64 ➔ Dikirim ke Server.
🛠 Cara Rubah Kalau Mau Diedit: Kalau tiba-tiba ada bug orang ngestuck setelah gambar 3, buka blok coding ConversationHandler di tengah-tengah bot.py. Pastikan rute VCG_IMAGE_C melompat ke VCG_IMAGE_D tanpa ada huruf salah ketik (typo).

7. Troubleshooting (Bengkel Dadakan)

Kasus 1

Bot cuma bengong pas di chat (Centang Dua Biru Saja)

  • Si Biang Kerok: Program Python di balik layar tiba-tiba terhenti atau tabrakan internet.
  • Dimana ceknya: Layar hitam command prompt komputermu.
  • Resep Obatnya: Jangan ubah coding. Hentikan (Pencet CTRL+C), dan nyalakan ulang sistemnya (python bot.py).
Kasus 2

KeyError: TELEGRAM_TOKEN merah di layar

  • Si Biang Kerok: Bot mau jalan menuju kantor tapi kehilangan kunci gembok Telegram.
  • Dimana ceknya: File yang namanya .env.
  • Resep Obatnya: Buka file tersebut pakai text editor. Tambahkan tulisan pastikan ada TELEGRAM_TOKEN=kode-bot-kamu tanpa salah spasi sedikitpun.

8. Playbook Maintenance (Buku Sakti Admin)

A

Mengubah Karakter Sifat AI (Update Prompt)

Misal AI dirasa terlalu resmi/kaku, kamu tidak usah membongkar satupun koding python! Cukup buka folder assets/inputs/, cari nama subjek yang diubah (Misal PR), lalu edit ganti perintah bahasa di teks file .md nya dengan Notepad. Selesai!

B

Mengisi/Menagih Koin Orang Lain

Kamu (sebagai Admin) dapat masuk ke bot, lalu ngetik /add_balance. Bot akan meminta ID angkanya si Budi dan berapapun kamu isikan otomatis saldonya nambah drastis di tangan Budi.

Peraturan: JANGAN PERNAH DI UBAH!

  • Jangan mengganti variabel konstan rahasia bernama ConversationHandler.END, kalau diganti sedikit, aplikasi akan lupa bagaimana cara mematikan sesi chat.
  • Jangan mengganti ejaan user_id di tab Model karena pondasi database akan roboh terbentur inkonsistensi tipe data lama dan baru.

9. Glossary (Kamus Singkat)

AI

Otak robot pintar yang gajinya dibayar per token / koin yang bertugas menilai tugas gambar dan tulisan.

API (KIE)

Jembatan supranatural atau pipa transparan yang membuang pertanyaan dan mendatangkan jawaban ke bot dari Google.

Database

Tempat penyimpanan keras seperti arsip brankas besi (tempat rahasia semua sisa koin berada).

Handle / Router

Polisi simpang empat di jalan bot Python. Tugasnya: bila Budi klik A, lempar Budi ke utara. Bila klik B, lempar ke Selatan.

Prompt

Kata-kata suci sakti yang dikirimkan ke dewa AI agar dia mau melakukan perkerjaan yang kita minta.

Log

CCTV atau buku dosa. Rekaman segalanya, agar sang ahli paham kenapa aplikasinya mendadak diam seribu bahasa.
