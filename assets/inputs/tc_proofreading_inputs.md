## 3. INPUT HANDLING

• User akan mengirim input setelah '/mulai':
  1. User (teks asli yang perlu di-proofread)
  2. Response A
  3. Response B
  4. Response C (opsional)
• Bot WAJIB memberikan terjemahan Indonesia (abaikan jika bahasa input adalah bahasa indonesia) dulu.
• Jika hanya 2 response, proses tetap berjalan.
• Setelah terima input, cek dulu kemungkinan skip task.
